# Agent Design Starter Kit

> A starting point for systematically designing, governing, and expanding an agent portfolio using the common agent contract.

## What This Starter Kit Contains

This starter kit provides the tools to design new agents rigorously and manage a growing portfolio:

1. A blank agent contract (ready to fill in)
2. An agent registry template
3. A design review checklist
4. A portfolio health dashboard template

## Who This Is For

Organizations that are:

- Ready to move beyond their first 1–2 agents
- Building a systematic approach to agent design and governance
- Managing multiple agents across different business domains
- Preparing to scale agent use across the organization

## Prerequisites

Before using this starter kit, you should have:

- [ ] At least one agent successfully deployed and reviewed
- [ ] A knowledge base structure in place
- [ ] A governance model operational (not just documented)
- [ ] An agent owner model established

## How to Use This Starter Kit

1. Use `1-agent-contract-blank.md` for every new agent design — complete it fully before development begins
2. Maintain `2-agent-registry.md` as agents are deployed — this is your organizational record
3. Apply `3-design-review-checklist.md` at the design review gate before any agent is approved
4. Use `4-portfolio-health-template.md` for quarterly portfolio reviews

## Files in This Starter Kit

| File | Purpose |
|------|---------|
| `1-agent-contract-blank.md` | Blank agent contract template |
| `2-agent-registry.md` | Registry for all deployed agents |
| `3-design-review-checklist.md` | Pre-deployment review criteria |
| `4-portfolio-health-template.md` | Quarterly portfolio health review |

## The Agent Design Process

```
Business need identified
        │
        ▼
Complete agent contract (1-agent-contract-blank.md)
        │
        ▼
Design review (3-design-review-checklist.md)
        │
        ▼
Build and test
        │
        ▼
Deploy and register (2-agent-registry.md)
        │
        ▼
Quarterly review (4-portfolio-health-template.md)
```
