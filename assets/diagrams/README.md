# assets/diagrams/

Place exported diagram assets here (rendered images from Mermaid or other tools).

- Source diagrams (`.mmd`) belong in `examples/diagrams/`
- This folder contains rendered/exported versions (`.png`, `.svg`) for use in presentations or external docs
- Use the same filename as the source `.mmd` file, with the appropriate image extension
- Example: `architecture-layers.mmd` → `architecture-layers.png`
