## Conventions

- Use descriptive filenames; prefer kebab-case for new files.
- Preferred formats: `.svg`, then `.png`.
- Include light and dark variants where applicable (e.g. `logo-light.png`, `logo-dark.png`).
- Do not commit logos that are proprietary or trademarked without authorization.

> The EmprendHEC logo is the brand mark of the reference organization and is included with authorization for use in this repository's documentation. Do not reuse it outside this context.
